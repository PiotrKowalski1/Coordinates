# Coordinates
Prosty program matematyczny, który oblicza pola i obwody figur matematycznych na podstawie wprowadzonych przez użytkownika współrzędnych danych figur. Program wykonuje obliczenie dla 3 prostych figur: prostokąta, trójkąta i koła. Dzięki zastosowaniu polimorfizmu jest łatwo rozbudowywalny i można dokładać nowe figury. W programie znajduję się klasa abstrakcyjna, będąca klasa nadrzędną dla klas wszystkich figur. Ponadto program wyposażony jest w mechanizm przechwytywanie błędów, gdyby użytkownik wprowadził zły format danych. Dzięki temu program nie przerywa działanie w nieoczekiwanym momencie.
